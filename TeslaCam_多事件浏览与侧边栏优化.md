# TeslaCam 多事件浏览与侧边栏优化

## Core Features

- 选择根目录并自动识别事件子目录

- 左侧现代侧边栏列出事件（日期_时间）

- 侧边栏搜索过滤事件

- 点击事件项后右侧合并同步播放多视角

- 保留并整合底部播放控制与时间滑条联动

- 空状态与异常过滤提示

- 侧边栏行高优化与自动播放稳定

- 搜索性能优化（防抖+后台过滤）
